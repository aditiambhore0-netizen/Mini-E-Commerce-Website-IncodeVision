# 🛒 NovaCart - Mini E-Commerce Website

A modern and responsive E-Commerce Website built using HTML, CSS, and JavaScript as part of the IncodeVision Internship Program. The project provides a complete online shopping experience with product browsing, cart management, wishlist functionality, coupon discounts, checkout, order history, invoice generation, and an AI-powered shopping assistant.

## 🌟 Live Features

### 🏪 Shopping Experience
- Browse featured products
- Product categories
- Product ratings
- Product pricing
- Modern UI design

### 🔍 Smart Search & Filter
- Search products instantly
- Filter products by category
- Dynamic product display

### 🛒 Shopping Cart
- Add products to cart
- Remove products from cart
- Real-time cart updates
- Automatic total calculation

### ❤️ Wishlist System
- Add products to wishlist
- Wishlist counter
- Local storage support

### 🎟️ Coupon System
Apply discount coupons:

| Coupon Code | Discount |
|------------|----------|
| ADITI20 | 20% OFF |
| SAVE10 | 10% OFF |

### 🌙 Dark Mode
- Toggle dark/light theme
- Theme saved using Local Storage

### 🤖 AI Shopping Assistant
Interactive chatbot that:
- Recommends products
- Suggests gaming accessories
- Suggests electronics
- Helps customers find products under budget

### 💳 Checkout System
- Customer Details Form
- Address Collection
- Phone Number Collection
- Order Placement

### 📦 Order Management
- Order History Page
- Order Details Storage
- Invoice Generation
- Download Invoice Feature

### 📱 Responsive Design
- Mobile Friendly
- Tablet Friendly
- Desktop Optimized
